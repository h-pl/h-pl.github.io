排版布局的新规则

语法 释义或用法 备注
旧的布局排版
Normal flow 文档流
新的布局排版
Flex和Grid 定义流 整体属性
子元素为复合属性 定义流与文档流的区别是影响子元素
display: inline-flex; 对外表现为块，看似block 弹性-Flex
display: inline-grid; 对外表现为行内，看似inline-block 网格-Grid
弹性布局-Flex 一维布局
flex-direction 主轴方向，row从左至右 row-reverse，反向
辅助轴，column，从上至下 column-reverse，反向
flex-wrap 是否换行
nowrap 不换行，default
wrap 有需要换行
wrap-reverse 有需要反向换行
order 数值越大有靠后，越小越靠前，可以取负值，不能取小数。 视觉上的次序
flex-grow 控制元素扩张 主轴必须存在空白
子元素default=0，不扩张
flex-shrink 控制元素收缩 主轴空间不足时
子元素default=1，且不能取负值。0 表示不收缩。
flex-basis 主轴方向尺寸-宽度或高度 元素在扩张、收缩前，主轴上占用的长度
对齐 前缀-后缀=6种组合:取值
前缀
justify- 主轴-横向对齐
align- 辅轴-纵向对齐
后缀
-content 所有子元素在父元素中如何摆放
-items 某行或列在某方向上如何对齐
-self 某个元素在某方向上如何对齐，即对于上一个属性的个别修正
6种组合的取值
stretch 拉伸至填满 flex布局，主轴上不能使用这个值
用flex-grow、flex-shrink来控制伸缩
center 某方向的中间
flex-start、flex-end 开始或结束 flex-start 是默认值
baseline 基于基线对齐 当前元素的底边位置就是基线
space-between space 空白进行合理分配，只有带 content 的属性才用。
between 两个子元素之间平均分配空白；
space-around around 环绕，两子元素间是两倍空白，而子元素和父元素间是一倍空白
space-evenly evenly 平均分配，所有元素空白统统相等。
网格布局-Grid 二维xy布局 子元素 display: subgrid;
先画出格子，再往里面放东西。 显式网格和隐式网格
网格线 网格中的每一条线
网格单元格 网格中的每一个单元格
网格区域 几个单元格组成的矩形
网格轨道 两个相邻网格线之间的空间 某一行或某一列
网格项 网格内子元素
grid- 网格布局固定的前缀
-template- 模板，显式网格
-auto- 自动，隐式网格
-rows 水平网格线的垂直间隔 行
-columns 垂直网格线的水平间距 列
网格定位简写
-start grid-column-start:2;grid-row-start:2; 坐标为网格线(2,2)右下
-end grid-column-end:2;grid-row-end:2; 坐标为网格线(2,2)左上
grid-template-columns
grid-template-rows
grid-auto-columns
grid-auto-rows
取值
repeat(6,100px) 6个100px的间隔
50px 30px 1fr 3fr 5fr 5个间隔:1个50px,1个30px,剩下空白9等分(各占1,3,5份)
[first-start] [] 表示网格线，里面是网格线的名字
span xxx 跨越某网格线
网格细节
grid-column-gap 网格线的粗细-间隔，default=0
grid-row-gap 网格线的粗细-间隔，default=0
grid-gap 上述两个的复合属性
grid-auto-flow 自动控制没有固定的元素
row
column
dense 密集的
grid-template-areas 画棋盘，填棋子 填名字，空为.
对齐 子元素并不会占满格子➡️具体排布靠对齐属性
justify 横向对齐
align 纵向对齐

网格布局

定位简写

-start 网格线右侧，-end 网格线左侧

.grid-item {
    grid-column-start: 1;
    grid-column-end: 3;
    grid-row-start: 2;
    grid-row-end: 4;
}
/* 简写为 */
.grid-item {
    grid-column-start: 1/3;
    grid-row-start: 2/4;
}

.grid-item {
    grid-column-start: 1;
    grid-column-end: span 1;
    grid-row-start: 2;
    grid-row-end: span 2;
}

areas 定位

.grid-box {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
grid-template-rows: 1fr 1fr 1fr;
grid-template-areas: 
". chess ."
". chess ."
".   .   .";
}
.grid-item {
min-height: 100px;
grid-area: chess;
}
